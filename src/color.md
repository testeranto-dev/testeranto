# Accessible Low-Blue Graph Visualization Palette

This theme is designed for **Color Universal Design (CUD)** to accommodate Protanopia, Deuteranopia, and Tritanopia while minimizing blue light exposure.

## Semantic State Mapping


| State          | Light Mode BG      | Light Mode FG | Dark Mode BG     | Dark Mode FG      |
| :------------- | :----------------- | :------------ | :--------------- | :---------------- |
| **canvas**     | `ivory`            | `black`       | `obsidian`       | `silver`          |
| **success**    | `oliveDarkSubtle`  | `oliveDark`   | `oliveDeep`      | `bluishGreen`     |
| **warning**    | `amberGoldSubtle`  | `amberDark`   | `amberDeep`      | `amberGold`       |
| **error**      | `deepOrangeSubtle` | `deepOrange`  | `deepOrangeDeep` | `vermillionLight` |
| **neutral**    | `charcoalSubtle`   | `charcoal`    | `charcoalDeep`   | `silver`          |
| **inProgress** | `rustSubtle`       | `rust`        | `rustDeep`       | `reddishPurple`   |
| **muted**      | `warmGreySubtle`   | `warmGrey`    | `warmGreyDeep`   | `warmGreyLight`   |
